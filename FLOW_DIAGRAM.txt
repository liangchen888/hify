│ ├─ 加载 Session (MySQL)                                                 │   │
│ ├─ 加载 Agent 配置 (Redis Cache)                                        │   │
│ ├─ 加载 ModelConfig (Redis Cache)                                       │   │

│ └───────────────────────────────────────────────────────────────────────┘   │

│ ├─ MySQL 回源: SELECT 最近 20 条消息                                   │   │

│ │                                                                       │   │

│ ├─ 从内存 MOCK_CHUNKS 提取片段                                          │   │
│ ├─ Mock 相似度排序 (真实: embedding + pgvector)                         │   │

│ └───────────────────────────────────────────────────────────────────────┘   │
└──────────────────────────┬──────────────────────────────────────────────────┘

│ │ System Message:                                                      │   │

│ │ buildToolSchemas(agentId)                                            │   │

│ └───────────────────────────────────────────────────────────────────────┘   │
└──────────────────────────┬──────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│ 🧠 LLM 第一轮调用 [大模型决策点 1]  T+100ms                                   │
│ ┌───────────────────────────────────────────────────────────────────────┐   │
│ │ ProviderAdapter.streamChat(provider, chatRequest, onChunk callback) │   │
│ │                                                                       │   │
│ │ 请求:                                                                │   │
│ │   POST https://api.openai.com/v1/chat/completions                   │   │
│ │   {                                                                  │   │
│ │     "model": "gpt-4",                                                │   │
│ │     "messages": [...],                                              │   │
│ │     "temperature": 0.7,                                              │   │
│ │     "max_tokens": 2048,                                              │   │
│ │     "tools": [                                                       │   │
│ │       {"type": "function",                                           │   │
│ │        "function": {                                                 │   │
│ │          "name": "check_refund_eligibility",                         │   │
│ │          "description": "...",                                       │   │
│ │          "parameters": {...}                                        │   │
│ │        }}                                                            │   │
│ │     ]                                                                │   │
│ │   }                                                                  │   │
│ │                                                                       │   │
│ │ 流式响应处理:                                                        │   │
│ │   while (chunk) {                                                    │   │
│ │     delta = extract_text(chunk)                                      │   │
│ │     onChunk(delta)  ◄─────┐                                         │   │
│ │   }                        │                                         │   │
│ └───────────────────────┬────┼─────────────────────────────────────┘   │
│                         │    │                                        │   │
│                         │    ▼                                        │   │
│                    ┌────────────────┐                                │   │
│                    │ SSE 推送给前端 │  event: {"type": "delta",     │   │

│                    └────────────────┘                                │   │
│                                                                       │   │
│ 响应分析:                                                            │   │

│     └─ 直接返回最终回答，跳过工具调用                                 │   │
│                                                                       │   │
│   ✓ 如果 finish_reason == "tool_calls" ◄─ [大模型主动决策]           │   │
│     └─ 返回工具调用指令                                               │   │
│                                                                       │   │
│ 假设本次返回: {"finishReason": "tool_calls",                         │   │
│              "toolCalls": [{                                         │   │
│                "id": "call_123",                                      │   │
│                "name": "check_refund_eligibility",                    │   │

│              }]}                                                      │   │
│ └───────────────────────────────────────────────────────────────────────┘   │
